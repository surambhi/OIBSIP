# WanderScape - Travel Landing Page

## Project Overview

WanderScape is a responsive travel landing page built using HTML and CSS.
This website showcases travel destinations, features, testimonials, and contact information.


## Features

* Fixed Navigation Bar
* Hero Section with CTA Button
* Popular Destinations Section
* Features Section
* Responsive Design
* Footer Section


## Tech Stack

* HTML5
* CSS3


## Folder Structure

WebDev-L1-LandingPage/
├── index.html
├── index.css
├── images/
└── README.md

## How to Run

1. Download project files
2. Open `index.html` in browser
